# OpenCog Unsupervised Language Learning project

## Dependencies
* Anaconda 3
* Numpy
* Pandas
* Scikit-Learn
* Jupyter notebook
* Matplotlib
* Cython
* SparseSVD
* PyTest

## Create virtual environment
```
conda env create -f environment.yml
```
## Run tests
```
cd ~/language-learning
source activate ull3
pytest
```
## Run Jupyter
```
cd ~/language-learning/notebooks
source activate ull3
jupyter notebook
```
